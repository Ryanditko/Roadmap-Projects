# Real-time Chat Backend (WebSockets)

## Idea
Build a chat system with real-time message delivery using WebSockets. Learn about persistent connections, message ordering, and presence tracking.

## Learning Objectives
- Implement WebSocket server
- Handle persistent connections
- Implement message routing
- Build presence tracking
- Handle disconnections gracefully

## Implementation Tips
- Use WebSocket framework (Socket.io, Websockets, etc.)
- Implement connection pooling
- Create message queue for reliability
- Add message persistence
- Implement presence tracking
- Create typing indicators
- Add user status updates
- Implement message search
- Create chat history
- Add read receipts
- Implement message reactions
- Create notification support
- Add moderator features
- Implement message encryption (optional)

## Key Challenges
- Connection reliability
- Message ordering across servers
- Presence accuracy
- Scalability with many connections
- Message persistence and recovery
- Network latency handling
