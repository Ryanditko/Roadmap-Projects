# Chat UI (WebSocket integration)

## Idea
Build a chat interface that connects to a WebSocket server. Learn about real-time communication, message handling, and user interactions.

## Learning Objectives
- Implement WebSocket connection
- Handle message sending/receiving
- Build message list UI
- Implement presence indicators
- Manage chat state

## Implementation Tips
- Create message input area
- Implement message list with auto-scroll
- Add user list with status
- Create typing indicators
- Add message timestamps
- Implement message search
- Create message reactions
- Add emoji support
- Implement file sharing UI
- Create user profiles on click
- Add message threading
- Implement read receipts
- Create notification support
- Add accessibility features

## Key Challenges
- Real-time message ordering
- Connection reliability
- Presence accuracy
- Message persistence
- Scalability with many users
