**Level:** Advanced

## Description

A Software as a Service (SaaS) system with paid plans and customer management.

## Features

- Registration and login with different permissions
- Subscription plans with recurring payments
- Team management with workspaces
- Public API for integrations
- Usage monitoring and metrics

## Learning Objectives

- Multi-tenant architecture
- Subscription billing
- Role-based access control (RBAC)
- API design and versioning
- Rate limiting and quotas
- Webhooks for integrations
- Usage metering
- Tenant isolation

## Implementation Tips

1. Isolate data by tenant (schema or database)
2. Implement billing with Stripe Subscriptions
3. Create permission system (RBAC)
4. Expose REST or GraphQL API
5. Implement rate limiting by plan
6. Monitor usage for billing and analytics
