**Level:** Advanced

## Description

A system to simulate financial market investments with real data.

## Features

- User registration and investor profile
- Simulated stock buying and selling
- Interactive charts showing investment performance
- Integration with market APIs for real data
- Rewards for successful predictions

## Learning Objectives

- Financial market APIs
- Real-time data streaming
- Financial charts (candlestick)
- Profitability calculations
- Portfolio management
- Risk analysis
- Market simulation
- Technical indicators

## Implementation Tips

1. Integrate APIs (Alpha Vantage, Yahoo Finance)
2. Implement WebSockets for real-time data
3. Create candlestick and line charts
4. Calculate indicators (RSI, MACD, moving averages)
5. Simulate portfolio with virtual balance
6. Add ranking of best investors
