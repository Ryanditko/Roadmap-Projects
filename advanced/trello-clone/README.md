**Level:** Advanced

## Description

A platform to manage tasks with an advanced drag-and-drop system.

## Features

- Create boards, lists, and task cards
- Drag and drop tasks between columns
- Comments and assignment of responsible parties
- Support for attachments and notifications
- Modern and responsive interface

## Learning Objectives

- Complex drag and drop
- Real-time collaboration
- Hierarchical data architecture
- Optimistic UI updates
- WebSockets for synchronization
- Permission system
- File upload
- Rich text editing

## Implementation Tips

1. Structure data (workspaces, boards, lists, cards)
2. Implement DnD with reordering and animations
3. Use WebSockets for real-time collaboration
4. Add permission system (admin, member, viewer)
5. Implement search across all cards
6. Add activity/logs for all actions

## Useful Resources

- CRDTs for collaboration
- Operational Transformation
- WebSocket best practices
- Drag and drop patterns

- Colored labels and tags
- Due dates and calendar
- Checklists within cards
- Custom fields
- Power-ups/integrations
- Board templates
- Time tracking
- Automations (Butler-like)
- Mobile app
- Offline support
